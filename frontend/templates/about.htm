About Sir Kothay

"Sir Kothay Achen" is a lightweight web application designed to make sharing messages simple and efficient. Whether you want to leave a quick note, share a thought, or send a message to someone, Sir Kothay provides a unique, shareable URL and a QR code for easy access.

How It Works

1. Create a Message

Write your message on the home page and save it.

2. Generate a Unique URL

Each message gets a dedicated page with a unique URL.

3. Share via QR Code

A QR code is automatically generated for quick sharing.

4. View the Message

Anyone with the link or QR code can view your message instantly.
